# test-wgpu — 体积云渲染 Demo

基于 **Rust + wgpu** 实现的实时体积云（Volumetric Clouds）效果。无任何贴图资源，云形、天空、地面、大气全部由着色器程序化生成。

v2 参考 [three-geospatial 的 London 演示](https://takram-design-engineering.github.io/three-geospatial/?path=/story/clouds-3d-tiles-renderer-integration--london)进行了大幅升级：**丁达尔效应（云隙光柱）**、**云层动态翻腾演化**、**egui 实时调参面板**。

![screenshot](2026-06-10-14-45-47.gif)

## 技术原理

### 渲染管线（v2）

```
[Pass 1] shader.wgsl  体积云+大气光线步进 → 离屏 HDR 纹理(Rgba16Float, 分辨率可缩放)
[Pass 2] blit.wgsl    双线性放大 → 交换链
[Pass 3] egui         参数面板叠加
```

### 光路合成模型

沿视线从近到远分段积分，能量守恒合成：

```
[段A 大气] → [云层] → [段B 大气(向下视线)] → [背景(天空/地面)]
result = Sa + Ta · ( S云 + T云 · ( Sb + Tb · bg ) )
```

1. **云形密度**：5 八度 FBM 值噪声 + 对比度锐化（云块分明）× 高度造型曲线（底平顶圆）− 覆盖率阈值 − 高频细节边缘侵蚀（云顶更碎）
2. **云自阴影**：每采样点向太阳做 6 步指数间距光照步进，Beer 定律衰减 + 多重散射近似（云底不死黑）
3. **丁达尔效应**：段 A/B 大气内散射步进中，每点计算"向太阳穿过整个云层的透射率"（5 采样），云遮挡处暗、云隙处亮，配合紧凑前向 Mie 相位瓣（g=0.80）形成从云后散开的光柱；透射率平方增强明暗对比
4. **银边**：云相位函数为 HG 前向（g 可调）混合后向散射，逆光云缘发亮
5. **动态演化**：风场水平平移 + evolution 参数驱动形状域垂直推进（翻腾感）+ 细节域独立时间相位（边缘持续翻滚）
6. **氛围**：太阳颜色随高度角变暖（低角度橙红）、地平线朝阳侧染色、ACES 色调映射
7. **地面云影**：地面直射光受云层透射率调制；阴影用单独的"羽化阈值"密度场计算——把云边界的硬阶跃展宽为 smoothstep 过渡，等效于对投影做空间模糊，半影宽度与暗度均可在面板调节
8. **抗条带**：所有步进起点逐像素抖动

### CPU 侧

`main.rs`：winit 0.30 + wgpu 25 + egui 0.32。每帧将 GUI 参数写入 Uniform 缓冲（160 字节），所有调参零延迟生效；渲染比例改变时重建离屏纹理。egui 默认字体不含中文，启动时从系统目录加载微软雅黑作为回退。

## 运行

```bash
cargo run
```

> 国内网络已在 `.cargo/config.toml` 中配置了项目级 rsproxy 镜像，不影响全局 cargo 配置。

## GUI 参数面板

| 分组 | 参数 |
|---|---|
| ☁ 云形 | 覆盖率、密度、细节侵蚀、消光系数、云底/云顶高度 |
| 🌀 动态流动 | 风速、风向、翻腾演化、时间流速（0 = 暂停） |
| ☀ 光照与太阳 | 太阳高度角/方位角、阳光强度、环境光、前向散射 g、后向散射混合 |
| 🌅 丁达尔效应 | 光柱强度、大气雾密度、光柱步进数 |
| 🌍 地面云影 | 云影强度、云影柔和度（羽化阈值实现半影软化） |
| 🖥 渲染质量 | 云步进数、光照步进数、渲染比例（性能/画质权衡）、曝光 |

面板内置「黄昏光柱预设」一键切换到低太阳强光柱场景；「重置默认」恢复初始参数。

**观察丁达尔效应的技巧**：把太阳高度角调低（5°~15°），方位角转到有大块云遮挡太阳的方向，增大「大气雾密度」和「光柱强度」——光柱在太阳被云缘半遮挡时最明显。

## 性能

- 默认渲染比例 0.75，2K 屏下约 30~60 FPS（视 GPU 而定）
- 帧率不足时优先降低「渲染比例」和「光柱步进数」；丁达尔效应是大头（每个大气采样点要做 5 次云密度求值）
- 噪声为纯 ALU 实时计算（未预烘焙 3D 纹理）。后续优化方向：噪声烘焙 3D 纹理、时域重投影（TAA 复用历史帧）、光柱半分辨率渲染
